# AdvanceRailway

AdvanceRailway は、マインクラフト内に敷設した鉄道ネットワーク（駅・路線・グループ）を構築・管理し、
[squaremap](https://github.com/jpenilla/squaremap) の Web マップ上に表示できる [Paper](https://papermc.io/) / Bukkit プラグインです。

## 主な機能

- **駅・路線・グループ** をコマンドで定義し、JSON としてプラグインのデータフォルダに保存します。
- 路線を **グループ（路線名・色）** でまとめられます。
- 駅・路線を **squaremap のライブマップ** にマーカー・ラインとして描画します（squaremap は任意の依存）。
- **経路検索**: 2 駅間（または現在地から駅）の最短経路を、レールと徒歩を組み合わせて求めます。
- **MineAuth HTTP API**: MineAuth プラグイン導入時、駅・路線・グループ・経路を JSON で取得できる読み取り専用エンドポイントを公開します。詳しくは [MineAuth HTTP API](/docs/mineauth-api) を参照してください。
